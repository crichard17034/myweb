The FPS Challenge Mode Revamp serves as a way to ramp up the rather lacking difficulty found in the FPS Microgame for Unity while also expanding on and implementing unused assets that were not present in the main Microgame.

TLDR: I didn't like base game, so I added mods.

Controls

  1. Arrow Keys/WASD: Walk
  2. Space Bar: Jump/Jetpack
  3. Shift: Sprint
  4. C Button: Crouch
  5. Left Click: Shoot
  6. Right Click: Aim
  7. Num Keys/Scroll Wheel: Change Weapon
